## Acolhimento!

Este é um projeto simples em C# desenvolvido para demonstrar os conceitos básicos de Programação Orientada a Objetos (POO), manipulação de strings e entrada/saída de dados no console.

### Sobre o Projeto

O programa solicita o nome de um usuário através do terminal, processa essa informação utilizando uma classe dedicada (Saudacao) e retorna uma mensagem de boas-vindas formatada.

Principais conceitos aplicados:

- Classes e Objetos: Uso da classe Saudacao para encapsular a lógica.

- Encapsulamento: Uso de campos privados e readonly.

- Interpolação de Strings: Uso do recurso $"" para formatar mensagens de forma limpa.

- I/O (Input/Output): Interação com o usuário via Console.

### Tecnologias Utilizadas

- Linguagem: C#

- Framework: .NET (Core / 5 / 6 / 7+)

- Paradigma: Orientação a Objetos
